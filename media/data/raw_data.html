Top 200 Tools for Learning

The Top Tools for Learning 2020 was compiled by Jane Hart from the results of the 14th Annual Learning Tools Survey, and released on 1 September 2020. For general information about the survey and this website, visit the About page. For observations and infographics of this year’s list, see Analysis 2020.

Below you will find a table ranking the top 200 tools, It also shows their place on 3 sub-lists PL100 | WL100 | Ed100, ie the Top 100 Tools for Personal Learning (PL100), the Top 100 Tools for Workplace Learning (WL100) and the Top 100 Tools for Education (Ed100). The shading shows where most popular usage is). Their tool categorization is also included. Click on a tool name to find out more about it.
